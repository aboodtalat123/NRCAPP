# نشر NRCAPP

هذا المشروع تطبيق ASP.NET Core Blazor Server، لذلك يحتاج استضافة Backend وليس استضافة ملفات static فقط.

## تشغيل Docker

```bash
docker build -t nrcapp .
docker run -p 8080:8080 nrcapp
```

ثم افتح:

```text
http://localhost:8080
```

## أول تشغيل رسمي

- التطبيق لا يضيف بيانات وهمية عند التشغيل.
- تبدأ قاعدة البيانات فارغة، ومن بوابة الدخول اختر **مؤسسة** ثم **إنشاء مؤسسة**.
- بعد إنشاء المؤسسة تنتقل مباشرة إلى لوحة المؤسسة لإضافة خطط التوزيع الرسمية.

مسار المواطن يعمل فقط لهوية موجودة فعلاً في قاعدة البيانات.

## ملاحظات نشر

- المنفذ داخل الحاوية هو `8080`.
- بدون Connection String يستخدم التطبيق SQLite محلياً في `App_Data/relief.db`.
- للإنتاج الحقيقي أضف Connection String باسم `ConnectionStrings__ReliefDb` على منصة الاستضافة.
